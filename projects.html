Group ChatBot App

Group ChatBot

This is my final project for NYCDA. It was built using javascript, HTML and CSS. It works by creating a group on the apps webpage that will initiate the group chat through SMS. When one responds to the inital text message it will be routed to the others in the group, essentially creating a geroup chat through one contact, the CHatBot.

I used Webtask.io to handle my Express server, Badwidth API to handle the SMS chat, and Firebase to handle the DB. Press the link above to give it a try.



Instagram Clone App

IG Clone App

This is another NYCDA Project. It was built using javascript, HTML, SQLite and CSS. It works as a clone to the Instagram App. It allows the user to create an account, upload photos, see a main feed of all users and follow other users to only see their photos in their feed. We used Firebase to handle the login and photo storage and SQLite for the DB. Please follow the instructions in the GitHub repo in order to download and initialize the app.
